Decision: command-result --json is retired and must not be advertised or accepted. Explicit exceptions remain in scope as non-result JSON surfaces: --log-format json diagnostic logging to stderr, authored JSON inputs such as bulk plans, Beads JSONL import input, local config JSON files, canonical .atelier-state content/projection internals, Mission Control projection JSON, and future API/projection surfaces. Automation should use focused human commands plus durable .atelier-state records/projections, not stdout JSON result contracts.
