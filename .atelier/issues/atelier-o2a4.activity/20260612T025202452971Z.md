Workspace policy direction: Atelier should not force branch/worktree ceremony by default. The default start path should support doing everything in the current checkout/main branch for small prototypes and single-person projects. Branch and worktree workflows should be configurable opt-ins for teams or orchestrators that need review flow, parallelism, isolation, or collaboration. Explicit worktree commands remain available for advanced/orchestrator management, but ordinary users should not have to learn them before using `atelier start <issue-id>`.
