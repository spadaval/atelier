## Description

Agent Factory planning guidance tells planners to write the target state clearly enough for workers and validators to understand finished. This issue owns the Agent Factory planning procedure wording, not product docs, command help, or examples except for direct links needed by that procedure.

## Outcome

The Agent Factory planning procedure tells planners to write a concrete target state first and avoid replacing the target state with broad process language. It removes or rewrites instructions that ask planners to pre-design proof systems for validators.
