## Description

Scoped history flags can become a second query language. Review `history --issue`, `--mission`, `--epic`, descendant scope, and filters against the complexity budget and surviving record/dashboard views.

## Outcome

History keeps only scoped modes that prove distinct value beyond bounded activity in `issue show`, `work mission`, evidence records, or validation records. Removed or folded scoped modes update help, docs, and tests; surviving modes have bounded default output and clear operator jobs.
