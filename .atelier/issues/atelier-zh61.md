## Description

Make Atelier the durable operating substrate for Agent Factory mission work. Scope: active mission focus, mission lifecycle guidance, Agent Factory skill and procedure updates, command enforcement for start/finish/validation/evidence, and scenario validation that agents can follow Atelier's next-action guidance without private chat context. Out of scope: launching or supervising live agent processes, retry queues, or replacing lightweight non-mission issue work with mission ceremony. Acceptance: Agent Factory guidance is Atelier-native; mission status/start/work/worktree commands provide clear next actions; configured validators enforce required evidence and closeout gates; waivers remain explicit and bounded; end-to-end validation proves an orchestrator can start a mission, delegate issue work, use branch/worktree helpers, attach evidence, resolve blockers, and close out.
