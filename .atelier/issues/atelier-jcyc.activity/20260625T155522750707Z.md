Color library decision: use anstyle for ANSI/style mechanics if the implementation needs a library, but keep Atelier's semantic StylePolicy/TextStyle as the public rendering boundary. Command and panel code should not call color-library APIs directly; panels request semantic styles from RenderContext, and StylePolicy maps those to anstyle while preserving terminal detection and NO_COLOR behavior.
