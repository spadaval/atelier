## Description

Issue and mission authoring guidance no longer teaches Evidence or Validation sections as universal boilerplate. This issue owns authoring templates, command help, and Agent Factory planning text that present default sections; examples are owned by the examples issue.

## Outcome

Mission and issue authoring surfaces no longer treat `Evidence` or `Validation` prose as default required sections for ordinary issue or mission authoring. Workers and validators produce proof records after doing checks; planners focus on the outcome.
